# Instagram-Bot-With-Selenium
## Purpose
This is a first of many web bots that will allow users to create a social gathering by automating engagement.
## How To Run
```
python Main.py username password hashtags,seperated,by,a,comma
```
## Future Updates

- Selenium 2
